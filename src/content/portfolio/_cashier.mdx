## Overview
      Cashier is a financial SaaS platform designed to help small and medium businesses manage their finances easily and efficiently. This is a personal project inspired by <a href='https://buildwithangga.com/' target='_blank'>BuildWithAngga</a> Class, Cashier offers an intuitive user interface and complete features to meet your financial needs.

### Role & Responsibility:

1. **User Interface Designer**
   - Creating a sitemap.
   - Designing a low-fidelity design (wireframe).
   - Creating a style guide.
   - Designing high-fidelity design and create UI components.
2. **Front-End Web Developer**
   - Slicing UI design to code using Bootstrap 5 and custom CSS.

### Tech Stack & Tools:

- Astro Js
- Bootstrap 5 with Custom CSS
- Figma
- VS Code
- GIT
