# Upgrade Astro

Update your projectâ€™s version of Astro to the latest version using your package manager, You can upgrade to Astro latest version by running the `@astrojs/upgrade` command:

```sh
npx @astrojs/upgrade
```

or by running the upgrade command for your package manager:

```sh
npm install astro@latest
pnpm upgrade astro --latest
yarn upgrade astro --latest
```
